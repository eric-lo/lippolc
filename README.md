# LIPP-OLC

A parallelized version of LIPP learned index using optimistic locking

## Compile & Run

```bash
mkdir build
cd build
cmake ..
make
```

Run example:

```bash
./build/example
./build/example_without_fmcd
./build/example_bulk_load
```

## Usage

See `src/examples/*` and `src/core/lipp.h` for detail.
